# TCMPPSDK

TCMPPSDK 是一个用于 iOS 平台的 Swift Package Manager 包。

## 要求

- iOS 11.0+
- Xcode 12.0+
- Swift 5.5+

## 安装

### Swift Package Manager

在 Xcode 中添加依赖：

1. 选择 File > Add Packages...
2. 输入仓库 URL：`https://github.com/xcodegou/TCMPPSDK.git`
3. 选择版本规则（建议使用 Up to Next Major Version）
4. 点击 Add Package

## 使用方法

```swift
import TCMPPSDK

// 初始化 SDK

// 使用 SDK 的其他功能
// ...
```
